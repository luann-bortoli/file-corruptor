# File Corruptor (JavaFX)

A simple JavaFX desktop application that corrupts files by modifying random bytes and saves a corrupted copy.

<img width="815" height="516" alt="image" src="https://github.com/user-attachments/assets/f8e87576-7401-49e7-b76a-b9848a5cce72" />

## Features
- Choose any file from your computer
- Select corruption intensity:
  - Low Noise
  - Data Drift
  - Byte Collapse
- Generates and saves a corrupted copy of the file

## Technologies
- Java
- JavaFX

## Running the project
1. Clone the repository
2. Open it in IntelliJ / Eclipse
3. Run the `Main.java` file
